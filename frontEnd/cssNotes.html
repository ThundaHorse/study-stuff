CSS 'Box Model' and its layout components

CSS box model is a rectangualr layout paradigm for HTML elements that consist of:

I'm aligned left
  • Content

    Content of the box, where text and images appear

  • Padding

    Transparent area surrounding the content (space between border and content)

  • Border

    Border surrounding the padding(if there is) and the content

  • Margin

    Transparent area surrounding the border (space between border and neighbording elements)

What is Sass?

Sass or Syntatically Awesome StyleSheets is a CSS preprocessor that adds elegance and power.

It allows use of variables, nested rules, mixins, inline imports, and more.

What is a Grid system in CSS?

Grid system is a structure that allows for content to be stacked both vertically and horizontally in consistent and easily mangable fashion.

Is comprised of 2 key components: rows and columns.

    Examples of Grid Systems:
  • Simple Grid
  • Bootstrap
  • Pure
  • Foundation

Explain meta tags in HTML

  • Meta tags ALWAYS go inside the head tag
  • Always passed as a name/value pair
  • Are not displayed on the page but intended for browser
  • Can contain information about character encoding, description, title of the document
